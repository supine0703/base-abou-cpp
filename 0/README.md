可以使用 `html` 语法加 `<br>` 进行换行 <br>也可以在文本可以在末尾加两个空格换行（建议这么做）  
使用两 `*` **字体加粗** (ctrl+b)，使用一 `*` *字体倾斜* (ctrl+i)，使用三 `*` ***加粗又倾斜*** (ctrl+i+b)
使用 n个 `` ` `` 包围可以标记代码块 使用三个以上 `` ` `` 包围可标记代码段，并可以加上语言进行语法高亮

```cpp
int main() {
    int a = 0;
    return 0;
}
```
想表达作为关键字的特殊字符，可以使用 `` ` `` 包围， 就像 `\` 也可以使用转义符 \\
